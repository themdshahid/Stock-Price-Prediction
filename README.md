Stock Price Prediction Web App :-- (Python, Streamlit, Keras)

Built an interactive stock-price forecasting app using Streamlit, LSTM deep-learning model, and Yahoo Finance API; implemented data preprocessing, moving averages, scaling, and model-based future price predictions.

Integrated live market data using yfinance and implemented technical indicators (MA 100/200/250).

Built an interactive dashboard with Streamlit to visualize price trends, moving averages, and model predictions.
